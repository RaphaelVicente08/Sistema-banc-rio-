# Perfil Profissional – Site Pessoal

Este repositório contém um site pessoal desenvolvido apenas com HTML. O objetivo do projeto é apresentar meu perfil profissional de forma simples, direta e organizada.

Objetivo do Projeto

O site reúne informações importantes sobre minha trajetória profissional, incluindo:
	•	Uma breve apresentação.
	•	Certificados em formato JPG.
	•	Links de contato e redes profissionais.

Estrutura do Projeto
	•	index.html
Página principal do site, contendo todas as seções organizadas e os certificados em imagem.
	•	/img
Pasta com os certificados em JPG e outras imagens utilizadas no site.

Características Principais
	•	Estrutura simples desenvolvida exclusivamente em HTML.
	•	Certificados exibidos diretamente como imagens, sem necessidade de download.
	•	Layout leve, fácil de carregar e compatível com qualquer navegador.
	•	Código limpo e fácil de editar, permitindo adicionar novos certificados ou informações quando necessário.
